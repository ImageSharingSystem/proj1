Installation

First, you have to install Oracle Instant Client 11g. This can be obtained at:

http://www.oracle.com/technetwork/database/features/instant-client/index-097480.html

Our project is implemented in JSP. You have to install Tomcat in order to run. Tomcat can be obtained at:

http://tomcat.apache.org/tomcat-9.0-doc/index.html

After set up Tomcat environment, type in the terminal:

admin@> starttomcat

When the Tomcat is running, you can enter our system.

If you want to login with your own user name and password, you need to sign up first.

If you already had a user name and password, input your user name and password and then click "Login".












Sign Up




If user wants to sign up for an account, user needs to click on the "Sign Up" button beside the "Log in" button.

For the sign up part, the system will ask user to input the user name, password, first name, last name, address, email and phone number.

If the user name is already existing in the database, you cannot use the same user name twice.












Main Page







If you enter a valid username and password, you will enter the main page. In the main page, there are several applications you can use.

On the top left, you can input the photo key words and then will return a list of photos. The details of searching result will be shown on the search page.

Below the "search photos", it is groups information. You can add or delete groups by inputting group name at the bottom.

After you adding group names, the added group name will be shown on the top. If you click on the group name, you will jump to the group page. The details of group page will be shown in the group page.

On the right part, you can see your uploaded photos. As well, you can upload photos by clicking "choose file" and then choose photos you want to upload by click on "Upload" button.

On the top right corner, there is a "Logout". If you click on, you will log out and jump to the login page.

On the bottom right, there is an "User Documentation". If you click on, you will see how to use this image sharing system step by step.

Search Page







In the search result page, the given photos are key words from searching.

If you want to see details of each photo, just click on that photo. Then it will give you details of that photo.

The details of photo page will be descripted later.

If you want to go back to main page, clicking on back button.













Personal Image Page







If you click your own picture on the main page, you will jump to this page which shows the details of this photo.

You can edit this photo's subject, place, time and description by inputting some information.

As well, you can set permission of this photo. "Private" means just you can see this photo. "Public" means everyone can see this photo. If you click on "Others", you have to choose which group you want to share with. Then click "save" button to save the changed information. Beside "save" button, there is a "delete" button which you can delete this photo. And "back" button is for going back to main page.













Friend Image Page







Friend image page is very similar to the personal image page. The only difference is you cannot change the information of the photo and you cannot set permission of this photo.

If you want to go back to main page, clicking on the "back" button.













Friend Page







On the left part of the friend page, you can see your friend detailed information.

On the bottom, you can set your friend into which group. Then clicking on "save" button to save the changes.

On the right part, the window displays your friend's uploaded photos. If you click on the photo, you will see details of the photo which is the above page.

If you want to go back to main page, clicking on "back" button at the bottom.













Group Page







When you go to group page, there are several columns which display different group names.

Under each group name, there are a list of names which belongs to that group.

If you click on the name, you will go to that name's friend page.

If you want to go back to main page, clicking on "back" button.
